# 🚀 TP 1 : Premiers pas avec Node.js - Backend JavaScript

Une application simple en Node.js avec Express permettant de gérer des utilisateurs (CRUD basique) avec stockage dans un fichier JSON.

---

## 📌 Fonctionnalités

- ✅ Ajouter un utilisateur
- ✅ Afficher tous les utilisateurs
- ✅ Afficher un utilisateur par ID
- ✅ Interface web simple (HTML/CSS/JS)
- ✅ API REST avec Express
- ✅ Lecture/écriture de fichiers avec `fs`

---

## 🛠️ Technologies utilisées

- Node.js
- Express.js
- HTML / CSS / JavaScript
- Module `fs` (File System)

---
